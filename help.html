EmulatorJS Controls Manual

This guide explains the default keyboard controls for Game Boy Advance (GBA), Super Nintendo (SNES), and Game Boy Color (GBC) when using EmulatorJS. HTML5-based games may have their own control schemes, which can differ from these defaults.

Pokemon and Zelda

  • A: Z
  • B: X
  • L: Q
  • R: E
  • Start: Enter
  • Select: V
  • D-Pad: Arrow Keys

Top Gear and NBA Jam

  • A: Z
  • B: X
  • X: A
  • Y: S
  • L: Q
  • R: E
  • Start: Enter
  • Select: V
  • D-Pad: Arrow Keys

Tetris

  • A: Z
  • B: X
  • Start: Enter
  • Select: V
  • D-Pad: Arrow Keys

Slope and Minecraft
